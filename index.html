What You'll Learn

In this class, students will learn the fundamentals of how to build a modern website and think like real front-end developers. We will dive deep into HTML and CSS and introduce how Javascript is used on a website. By the end of the class, students will have completed a website that renders on both the web and on a mobile device.

Spring Term: Frontend

Front-end concepts

  • HTML / CSS / Javascript - How do they work together?
  • HTML - learning to build the blueprint of an app or site
  • CSS - bringing your HTML to life through styling
  • Pre-processors - understanding how they work and why people use them
  • Component-driven development
  • A mobile-first approach to styling
  • Flexbox - putting all your components into place
  • UX and accessibility principles
  • An introduction to Javascript
